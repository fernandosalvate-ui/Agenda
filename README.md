# 📱 Meu Dia — Lembretes

App de lembretes diários com notificações nativas no Android (PWA).

---

## 🚀 Como publicar no GitHub Pages (gratuito)

### Passo 1 — Criar conta no GitHub
1. Acesse **github.com**
2. Clique em **Sign up**
3. Preencha: nome de usuário, e-mail, senha
4. Confirme o e-mail

### Passo 2 — Criar um repositório
1. Clique no **+** (canto superior direito) → **New repository**
2. Nome: `meudiaapp`
3. Marque **Public**
4. Clique em **Create repository**

### Passo 3 — Fazer upload dos arquivos
1. Na página do repositório, clique em **uploading an existing file**
2. Arraste estes 3 arquivos:
   - `index.html`
   - `manifest.json`
   - `sw.js`
3. Clique em **Commit changes**

### Passo 4 — Ativar o GitHub Pages
1. Vá em **Settings** (aba do repositório)
2. No menu lateral: **Pages**
3. Em "Branch", selecione **main** → pasta **/ (root)**
4. Clique em **Save**
5. Aguarde ~1 minuto. Vai aparecer o link:
   👉 `https://SEU_USUARIO.github.io/meudiaapp`

---

## 📲 Instalar como app no Android

1. Abra o link acima no **Chrome** do Android
2. Toque no menu (3 pontinhos) → **Adicionar à tela inicial**
3. Confirme → o ícone aparece na tela inicial
4. Abra o app e toque em **Ativar notificações**

Pronto! O app funciona como nativo, com notificações no horário certo. ✅

---

## 🔔 Como funcionam as notificações

- Quando você ativa, o app agenda uma notificação para cada atividade com horário
- As notificações chegam mesmo com o app fechado (enquanto o Chrome estiver rodando)
- Ao adicionar uma nova atividade, a notificação é agendada automaticamente
- Ao reiniciar o dia, todas as notificações são reagendadas

---

## ⚠️ Observações

- **Android + Chrome**: funciona perfeitamente
- **iOS (iPhone)**: suporte parcial a partir do iOS 16.4 com Safari
- O app salva tudo localmente no celular (sem servidor, sem cadastro)
